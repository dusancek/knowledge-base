# React Knowledge Base

This project is a Vite + React app that turns local Markdown files into a browsable document viewer.

## Current features

- Nested folder navigation.
- Search across titles, paths, and Markdown content.
- GitHub-flavored Markdown rendering.
- Notion-like reading width and quiet visual style.
- Mobile-friendly sidebar behavior.

## Useful commands

```sh
npm run dev
npm run build
npm run preview
```
