# Writing Notes

Good knowledge-base notes are usually short, findable, and easy to connect to other notes.

## Suggested pattern

Start each note with:

- A clear title.
- A short summary.
- The important details.
- Links to related files or external references.

## Example note template

```md
# Topic Name

One-paragraph summary of the thing.

## Details

- Important point
- Important point

## Links

- [Related note](../some-folder/related-note.md)
```
