# Personal Knowledge Base

This is a local Markdown knowledge base. Add files anywhere inside the `docs/` folder and the app will pick them up in the left navigation.

## How it is organized

The sidebar mirrors your directory structure:

- Folders become expandable navigation groups.
- `.md` files become readable pages.
- The first `# Heading` in a file becomes the page title.

## Quick editing loop

1. Create or edit a Markdown file in `docs/`.
2. Save the file.
3. The app refreshes and renders the page.

> Keep notes small, link related ideas, and let the folder structure stay boring.

## Markdown support

GitHub-flavored Markdown works, including tables, task lists, code blocks, links, and blockquotes.
